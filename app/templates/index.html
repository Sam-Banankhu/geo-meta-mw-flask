Welcome to the Geo-Meta Project

About the Project

The Geo-Meta project is a location-based API platform that provides developers with access to geographic metadata of various regions, districts, cities, and more in Malawi. It aims to simplify the retrieval of location data and empower developers to build innovative applications leveraging this information.

Available Endpoints

  • /api/districts - Retrieve a list of all districts
  • /api/residential_areas - Retrieve a list of all residential areas
  • /api/traditional_authorities - Retrieve a list of all traditional authorities
  • /api/villages - Retrieve a list of all villages
  • /api/constituencies - Retrieve a list of all constituencies
  • /api/wards - Retrieve a list of all wards
